# Stage 2 Current Status

At the end of Stage 2, the project had:

- a larger real-Kepler training pipeline
- saved model files on disk
- a more realistic training baseline
- a first app prototype
- a better understanding of how the project would scale

Stage 2 answered the question:

"Can the same simple baseline pipeline work on a larger amount of real Kepler data and be reused in an app?"

The answer was yes. Stage 2 made the project more practical and helped turn it from a classroom-style prototype into a reusable system.
